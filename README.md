# MSI360 Client

Client-only React/TypeScript frontend for the MSI360 prototype.

The app renders questions from hard-coded client catalog data, keeps answers in browser
memory, runs client-side routing/scoring, and generates the English MSI360 PDF report in
the browser.

## Setup

```bash
npm install
npm run dev
```

The client runs at `http://localhost:5173`.

The server repo is not used by this version.
